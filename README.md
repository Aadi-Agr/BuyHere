# Ecommerce

HOW TO RUN PROJECT
##
Frontend
```bash
cd frontend
npm install
```
 Backend
 ```bash
cd backend
npm install
```
In Complete Project
 ```bash
npm run dev
```
have to initialize .env file and make keys like
1. PORT="like 5000"
2. MONGO_URI="url used to describe mongodb"
3. NODE_ENV="can be anything"
4. JWT_SECRET="any secreat used in jwt"
5. PAYPAL_CLIENT_ID="ID of paypal of the client"
